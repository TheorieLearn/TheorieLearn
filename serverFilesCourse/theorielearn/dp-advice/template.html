General DP Problem-Solving Advice
Once we have determined a subproblem definition, we want to create a recursive definition for this subproblem definition. In order to do so, we should:
  • Consider a choice that we need to make and create exhaustive cases based on that choice
  • For each case, figure out an expression representing the "value" of that case; this expression will often involve smaller subproblems
  • Figure out how to use the separate cases to evaluate the value of the current subproblem

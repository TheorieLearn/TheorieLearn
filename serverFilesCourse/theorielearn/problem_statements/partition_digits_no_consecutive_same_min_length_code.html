You are given a sequence of digits $A[1..n]$ where each digit is between 1 and 9 (inclusive). You are asked to insert $+$ signs in between the digits to partition them into terms which will be added together. The length of each term in the summation must be either 1, 2, or 3 digits. Furthermore, no two consecutive terms in the summation can have the same length.

What is the minimum sum that can be achieved under these constraints?

Example

If $A = 9212121921$, then the minimum sum that can be achieved is $9+21+2+12+1+92+1=138$.
  • Note that $9+2+1+2+1+2+1+9+2+1 = 30$ achieves a smaller sum, but this is not allowed because there are mulitple consecutive 1-digit terms.
  • Similarly, $9+21+21+21+9+21=102$ is not allowed because there are two consecutive 2-digit terms.

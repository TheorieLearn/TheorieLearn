• $\textbf{Lemma 1: } w \bullet \varepsilon = w \text{ for all strings }w.$
  • $\textbf{Lemma 2: } |w \bullet x| = |w| + |x| \text{ for all strings } w \text{ and } x.$
  • $\textbf{Lemma 3: } (w \bullet x) \bullet y = w \bullet (x \bullet y) \text{ for all strings } w, x \text{ and } y.$

The remainder of this section in the GPS will be using the prefix definition of the subproblem, that is: $$isInA(i) \text{ is true if and only if } s[1 .. i] \in A.$$ Defining a subproblem is no good if we don't know how to call it to obtain our final answer! First, we should specify the domain of our function. $isInA()$ evaluates whether a prefix of a string $s[1..n]$ is in $A$. This function takes in an integer $i$, that represents the end index of the prefix. What possible values could $i$ take on? (Remember that the empty string is a prefix of $s$, represented by $s[1..0]$.)

English Definition
Let $MinCost(i, j)$ denote the minimum possible cost of traveling from mile $i$ to mile $n$ (including the cost of the hotel at mile $i$) with $j$ remaining coupons.

Recursive Definition
$$ \begin{equation} MinCost(i, j)=\begin{cases} 0 & \text{if $i=n$}\\ HotelCosts[i] & \text{else if $i=n - 1$ and $j = 0$}\\ 0 & \text{else if $i = n - 1$ and $j > 0$}\\ \min \begin{cases} HotelCosts(i) + MinCost(i + 1, j)\\ HotelCosts(i) + MinCost(i + 2, j) \end{cases} & \text{else if $j = 0$}\\ \min \begin{cases} HotelCosts(i) + MinCost(i + 1, j)\\ HotelCosts(i) + MinCost(i + 2, j) \\ MinCost(i + 1, j - 1) \\ MinCost(i + 2, j - 1) \end{cases} & \text{otherwise} \end{cases} \end{equation} $$

We now need to determine the evaluation order for our memoization structure. In what order do we have to evaluate each subproblem in order to make sure that all of its dependencies are satisfied? Because we have two parameters, we now need to iterate over both $i$ and $j$.

  • Why do we need to iterate over $j$? That's because we don't know how many coupons Momo will use up until a certain mile marker $i$ in the trip, so we need to make sure we cover all possible $j$s for the coupon count.

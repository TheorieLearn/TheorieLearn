English Definition
Let $MinCost(i, j)$ denote the minimum possible cost of traveling from mile $i$ to mile $n$ (including the cost of the hotel at mile $i$) with $j$ remaining coupons.

Recursive Definition
$$ \begin{equation} MinCost(i, j)=\begin{cases} 0 & \text{if } i = n\\ HotelCosts[i] & \text{else if } i = n - 1 \text{ and } j = 0\\ 0 & \text{else if } i = n - 1 \text{ and } j > 0\\ \min \begin{cases} HotelCosts(i) + MinCost(i + 1, j)\\ HotelCosts(i) + MinCost(i + 2, j) \end{cases} & \text{else if } j = 0\\ \min \begin{cases} HotelCosts(i) + MinCost(i + 1, j)\\ HotelCosts(i) + MinCost(i + 2, j)\\ MinCost(i + 1, j - 1)\\ MinCost(i + 2, j - 1) \end{cases} & \text{otherwise} \end{cases} \end{equation} $$

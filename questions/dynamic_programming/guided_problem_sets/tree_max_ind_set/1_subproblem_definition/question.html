Since our structure is a tree, this lends itself quite naturally to a recursive definition based on the subtree rooted at any particular vertex.

We will use the following subproblem definition for the rest of the question:

Define $MaxFun(v, o)$ to be the maximum total fun values of a subset of employees chosen from the subtree rooted at $v$ where $o$ represents whether $v$ is included or not.

Let $MaxFun(v, o)$ be the maximum amount of fun achievable from the subtree rooted at $v$ given that we will include $v$ if $o = \text{Include}$ and exclude $v$ if $o = \text{Exclude}$.

From the previous question, we have two exhaustive cases representing the value of $MaxFun(v, o)$:
  1. If $o = \text{Include}$ → [fun from vertex $v$] + [Maxmimum fun of subtrees excluding children of $v$]
  2. If $o = \text{Exclude}$ → [Maximum fun of subtrees either including or excluding childrens of $v$]
We will now give an expression for the value of each of these cases. Keep in mind how you can use the subproblem definition of $MaxProfit$ when defining these expressions.

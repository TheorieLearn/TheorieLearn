English Definition
Let $MaxFun(v, o)$ be the maximum amount of fun achievable from the subtree rooted at $v$ given that we will include $v$ if $o = \text{Include}$ and exclude $v$ if $o = \text{Exclude}$.

Recursive Definition
$$ \begin{equation} MaxFun(i)=\begin{cases} Fun[v] + \underset{\forall c \in Adj[v]}{sum}MaxFun(c, Exclude)& \text{if o = Include}\\ \underset{\forall c \in Adj[v]}{sum} max\{MaxFun(c, Include), MaxFun(c, Exclude)\} & \text{otherwise} \end{cases} \end{equation} $$ (We will define the empty sum as $0$ to account for the base case)

Constructing Vertices in G'

Now, after noting what information we should track, we need to formally define the set of vertices in the new graph $G'=(V',E')$. We define $V'$ as the product of four sets:
  • $V$: the vertices of the original graph $G$
  • $R=$ $\{0, 1, \dots, n\}$ $\{0, 1\}$ $\{count\}$ ,
    representing if you have traversed a red vertex how many red vertices you have traversed the number of red vertices left to be traversed in the graph
  • $B=$ $\{0, 1, \dots, n\}$ $\{0, 1\}$ $\{count\}$ ,
    representing if you have traversed a blue vertex how many blue vertices you have traversed the number of blue vertices left to be traversed in the graph
  • $G=$ $\{0, 1, \dots, n\}$ $\{0, 1\}$ $\{count\}$ ,
    representing if you have traversed a green vertex how many green vertices you have traversed the number of green vertices left to be traversed in the graph
Additionally, since the path can start at any vertex, you should add a new start vertex $s$ to your new graph.

Mathematically, we define $V' =$ $(V\cup \{s\})\cup R\cup B\cup G$ $(V \times R \times B \times G) \cup \{s\}$ $(V\cup \{s\})\cap R\cap B\cap G$ $(V\times \{s\})\cup R\cup B\cup G$ $(V\times \{s\})\times R\times B\times G$ $(V\times \{s\})\cap R\cap B\cap G$

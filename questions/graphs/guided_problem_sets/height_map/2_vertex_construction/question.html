Constructing Vertices in G'

Now, after noting what information we should track, we need to formally define the set of vertices in the new graph $G'=(V',E')$. We define $V'$ as the product of three sets:
  • $R=$ {0, 1, ..., n} {0, 1} {NotSeen, Seen} , representing the row (first) index of the Elevation map how many positions in the Elevation map you have traversed if you have seen the peak in the Elevation map
  • $C=$ {0, 1, ..., n} {0, 1} {NotSeen, Seen} , representing the col (second) index of the Elevation map how many positions in the Elevation map you have traversed if you have seen the peak in the Elevation map
  • $D=$ {up, down} {0, 1, ..., max height} {0, 1, ..., number of positions} , representing uphill or downhill portion of the climb your current height how many mountain positions you traversed upwards or downwards
More concisely, we define $V'$ = R ∪ C ∪ D R x C x D R ∩ C ∩ D

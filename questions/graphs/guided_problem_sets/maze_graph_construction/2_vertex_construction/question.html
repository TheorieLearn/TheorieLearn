Constructing Vertices in G'

Now, after noting what information we should track, we need to formally define the set of vertices in the new graph $G'=(V',E')$. We define $V'$ as the product of three sets:
  • $V$: the vertices of the original graph $G$
  • $H=$ {0, 1, ..., |E|} {SeenOnce, SeenTwice} {NotSeen, Seen} , representing the number of hieroglyphics seen whether or not you have traversed an edge containing mysterious hieroglyphics. the number of hieroglyphics in the maze the number of edges in G without hieroglyphics
  • $C=$ {Full} {Full, Empty} {Full, RechargeNow, Empty} {0, 1, ..., |E|} , representing the current charge of the invisibility device the number of charging stations in the maze the number of high-risk edges that have been traversed
More concisely, we define $V'$ = V ∪ H ∪ C V x H x C V ∩ H ∩ C

We've defined the vertices $V'$ in $G'$. We now need to define the edges $E'$, using the existence and color of any given edge in $G$. The vertices in $G'$ look like $(v,i)$, where $i \in \{0,1,2\}$. Again, based on the idea that we are using $i$ to encode *the number of edges traversed so far modulo 3*, we can define $E'$ as follows: For every edge $e=u\to v$ in $E$ :
  • If $e$ is red, then an edge exists from (u,0) (u,1) (u,2) to (v,1) (v,0) (v,2) in $E'$.
  • If $e$ is white, then an edge exists from (u,1) (u,0) (u,2) to (v,2) (v,0) (v,1) in $E'$.
  • If $e$ is blue, then an edge exists from (u,2) (u,0) (u,1) to (v,0) (v,1) (v,2) in $E'$.

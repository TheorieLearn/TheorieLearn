Constructing Vertices in $G'$

Now, after noting what information we should track, we need to formally define the set of vertices in the new graph $G'=(V',E')$. We define $V'$ by composing these sets:

  • $V$: the vertices of the original graph $G$
  • $S=$ {0, 1, ..., |V|} {+, -} {0, 1, ..., |E|} ,
    representing the coefficient of the next edge weight in the alternating length how many vertices you have traversed in total the fewest possible number of vertices left to be traversed before reaching t

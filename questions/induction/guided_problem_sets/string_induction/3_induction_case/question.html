$\text{Prove that }|w^R| = |w| \text{ for every string } w.$
Now, it's time for the inductive case. In the inductive case, we suppose that *$\textbf{w = ax}$* We are trying to come to the conclusion that $|w^R| = |w|$ from $w = ax$. $|w^R| = |(ax)^R| \quad$ (because $w = ax$) $ = |x^R \bullet a| \quad$ (by definition of $^R$) = $|x^R| + |a| \quad$ (by Lemma 2) = $|x| + |a| \quad$ (by Induction Hypothesis) = $|ax| \quad$ (by Lemma 2) = $|w|$ (because $w=ax$) $|w^R| = |(x \bullet a)^R| \quad$ (by definition of $^R$)

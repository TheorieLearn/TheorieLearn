As a reminder, our overarching goal has been to prove the following language is undecidable:

$$ \text{ExceptPalindrome} ~ := \{\langle M \rangle ~ | ~ M ~ \text{rejects all palindromes and accepts all other strings}\} $$

From the previous question, we now have the following correct reduction from $\text{Halt}$ to ${\text{ExceptPalindrome}}$:

$$ \begin{align*} &\underline{\textbf{DecideHalt}(\langle M,w\rangle):}\\ &\text{Encode the following algorithm} ~ M': \\ &\qquad\underline{M'(x):}\qquad\qquad\quad\\ &\qquad\quad\text{Run} ~ M ~ \text{on input} ~ w\\ &\qquad\quad\text{return} ~ x ~ \text{is not a palindrome}\\ &\text{if} ~ \textbf{DecideExceptPalindrome}(\langle M' \rangle):\\ &\quad\text{return True}\\ &\text{else:}\\ &\quad\text{return False}\\ \end{align*} $$

All that's left is to show our reduction is correct. To do this, we need to show the algorithm $M$ halts on the given input $w$ if and only if $\textbf{DecideHalt}$ returns $\text{True}$.

In this question, we will prove the forward direction of the "if and only if" claim. In other words, we want to prove $$ M ~ \text{halts on} ~ w \implies \textbf{DecideHalt}(\langle M,w \rangle) ~ \text{returns True} $$

Suppose $M$ halts on input $w$. Then $M'$ accepts every input string $x$ that is not a palindrome, and rejects every input string $x$ that is a palindrome. So $M'$ is in the language $\text{ExceptPalindrome}$. So $\textbf{DecideExceptPalindrome}$ accepts the encoding $\langle M' \rangle$. So $\textbf{DecideHalt}$ correctly accepts the encoding $\langle M,w \rangle$. Suppose $M$ does not halt on input $w$. Then $M'$ accepts every input string $x$. In particular, $M'$ does not accept any non-palindromes, nor does it reject any palindromes. Then $M'$ diverges on every input string $x$. So $M'$ is not in the language $\text{ExceptPalindrome}$. So $\textbf{DecideExceptPalindrome}$ rejects the encoding $\langle M' \rangle$. So $\textbf{DecideHalt}$ correctly rejects the encoding $\langle M,w \rangle$.

As a reminder, we're trying to prove the following language is undecidable: $$ \text{AcceptPalindrome} ~ := \{\langle M \rangle ~ | ~ M ~ \text{accepts at least one palindrome}\} $$ And we're using the following reduction: $$ \begin{align*} &\underline{\textbf{DecideHalt}(\langle M,w\rangle):}\\ &\text{Encode the following algorithm} ~ M': \\ &\quad\begin{bmatrix} \underline{M'(x):}\qquad\qquad\quad\\ \text{Run} ~ M ~ \text{on input} ~ w\\ \text{return True} \quad\quad~~~ \end{bmatrix}\\ &\text{if} ~ \textbf{DecideAcceptPalindrome}(\langle M' \rangle):\\ &\quad\text{return True}\\ &\text{else:}\\ &\quad\text{return False}\\ \end{align*} $$

We need to show the algorithm $M$ halts on the given input $w$ if and only if $\textbf{DecideHalt}$ returns $\text{True}$.

In this question, we will prove the reverse direction of the "if and only if" claim. This time, we will prove the contrapositive. In other words, we want to prove $$ M ~ \text{diverges on} ~ w \implies \textbf{DecideHalt}(\langle M,w \rangle) ~ \text{returns False} $$ This will complete the proof, and allow us to conclude that the algorithm $\textbf{DecideAcceptPalindrome}$ cannot exist.

Suppose $M$ does not halt on input $w$. Then $M'$ diverges on every input string $x$. In particular, $M'$ does not accept any palindromes. So $\textbf{DecideAcceptPalindrome}$ rejects the encoding $\langle M' \rangle$. So $\textbf{DecideHalt}$ correctly rejects the encoding $\langle M,w \rangle$. Suppose $M$ does not halt on input $w$. Then $M'$ accepts every input string $x$. In particular, $M'$ accepts the palindrome "ILLINILLI". So $\textbf{DecideAcceptPalindrome}$ accepts the encoding $\langle M' \rangle$. So $\textbf{DecideHalt}$ correctly accepts the encoding $\langle M,w \rangle$.

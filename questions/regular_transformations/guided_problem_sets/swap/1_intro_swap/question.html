For regular language transformation problems, it's important that you fully understand the transformation in question. A good first step is to think about a very simple language and consider what strings are accepted by the transformed language. Consider the following language transformation: $\mathsf{SWAP}(L) := \{\mathsf{swap}(w) \mid w \in L \}$, where $\mathsf{swap}(w)$ is defined recursively as $$ \mathsf{swap}(w) = \begin{cases} w &\text{if } |w| \leq 1\\ ba \bullet \mathsf{swap}(x) &\text{if } w = ab \bullet x \text{ for some } a, b \in \Sigma \text{ and } x \in \Sigma^* \end{cases} $$
Suppose that $L = \{ 0001 \}$. What string is contained in $\mathsf{SWAP}(L)$?
Suppose that $L = \{ 10101 \}$. What string is contained in $\mathsf{SWAP}(L)$?
Suppose that $L = \{ 111000 \}$. What string is contained in $\mathsf{SWAP}(L)$?
